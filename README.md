## рҹ“ұ What is Snatch?

**Snatch** is a free, open-source Flutter app that lets you download videos, audio, and shorts from the most popular platforms вҖ” without watermarks, without ads, and without any subscription.

> Built for people who believe your downloads should belong to you.

---

## вңЁ Features

### рҹҺ¬ YouTube
- Download videos in **360p / 720p / 1080p**
- Extract **audio only** (MP3 / M4A)
- Download **YouTube Shorts**
- Saves directly to your **Gallery & Music Library**
- Real-time **download progress bar**

### рҹҺө TikTok
- Download TikTok videos **without watermark**
- Paste URL вҶ’ instant download
- Saves to gallery

### рҹ“ё Instagram
- Download **Reels, Posts, Stories**
- Photos **and** videos supported
- **Zero watermark**
- Saves to gallery

## рҹӣ  Build from Source

### Prerequisites

| Tool | Version |
|------|---------|
| Flutter | `>=3.10.0` |
| Dart | `>=3.0.0` |
| Android SDK | API 29+ |

### Steps

```bash
# 1. Clone the repo
git clone https://github.com/Afaqraza12/Snatch-video-Downloader.git
cd Snatch-video-Downloader

# 2. Install dependencies
flutter pub get

# 3. Run on device/emulator
flutter run

# 4. Build release APK
flutter build apk --release
```

APK will be at: `build/app/outputs/flutter-apk/app-release.apk`

## рҹ“Ұ Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | Flutter (Dart) |
| YouTube extraction | `youtube_explode_dart` |
| HTTP & downloads | `dio` |
| State management | `provider` / `riverpod` |
| Gallery save | `media_store_plus` |
| Permissions | `permission_handler` |
| Storage | `path_provider` |
| Notifications | `fluttertoast` |

## рҹ”җ Permissions Required

```xml
INTERNET
READ_EXTERNAL_STORAGE
WRITE_EXTERNAL_STORAGE
READ_MEDIA_VIDEO
READ_MEDIA_AUDIO
READ_MEDIA_IMAGES
```

All permissions are used solely for downloading and saving media to your device. No data is collected or shared.
